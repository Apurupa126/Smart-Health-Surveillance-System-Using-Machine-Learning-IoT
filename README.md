# Smart Health Surveillance System Using Machine Learning & IoT

## Overview
An IoT-enabled healthcare monitoring system that tracks heart rate, SpO₂, and body temperature in real time using ESP32 and healthcare sensors. The system includes fingerprint-based authentication for secure patient identification.

## Technologies
- ESP32
- MAX30100
- DSB1800
- Python
- Django
- Machine Learning
- IoT

## Features
- Real-time health monitoring
- Machine learning-based analysis
- Secure patient identification
- Smart healthcare dashboard
